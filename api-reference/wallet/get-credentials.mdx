---
title: "GET /api/wallet/:did/credentials"
description: "List all credentials stored in a holder wallet"
---

```http
GET /api/wallet/did:alyra:z6Mkf.../credentials HTTP/1.1
Authorization: Bearer <JWT>
X-User-DID: did:alyra:z6Mkf...
```

## Path Parameters

| Param | Description |
|-------|-------------|
| `did` | Holder DID. Supports base64/base58 and will be normalized by the backend. |

## Response

`200 OK`

```json
{
  "success": true,
  "credentials": [
    {
      "id": "cred-123",
      "credential_preview": {
        "full_name": "Alice Anderson",
        "employee_id": "EMP-001"
      },
      "status": "active",
      "ipfs_hash": "Qm...",
      "on_chain": true,
      "issued_at": "2025-10-25T19:35:00Z"
    }
  ],
  "count": 1
}
```

> Legacy credentials without `credential_preview` are automatically normalized with fallback attributes. Use pagination in future releases (coming soon).
